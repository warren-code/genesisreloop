# Testimonial Images

This directory contains profile images for testimonials featured in the ReLoop platform.

## Image Requirements

- Square aspect ratio
- Recommended size: 200x200px
- Format: jpg or png
- File naming convention: {first-name-lowercase}.jpg (e.g., sarah.jpg)
